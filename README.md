# MainlineAgent




## idee de base :

Création d'un site pour tout le monde (anglais, francais, chinois) faire découvrir l'investissement dans des entreprises.

### produit jeu (agentgame)

Le produit jeu a pour but de développer des compétences de compréhension de rapport financier à l'utilisateur sa liberté d'analyse.

### produit notation (momoia)

Le but est de faire une ia évolutive en fonciton de ce que l'utilisateur recherche nous lui présentons à la fin un repport de l'entreprise choisis pour lui montrer les forces et les risques liées a un tel investissement.

### produit cours

des petits cours sous forme de doc pdf

### produit home 

Le produit home est la page d'accueil du site qui se décompose en 5 parties
accueil (momoia)
citation
presentation entreprise
presentation (agent game)

### produit annexe

- Barre de progression + Badge + Défi hebdo
- Blog type histoire
- Vidéos Youtube
